# **App Name**: 휙휙 메모

## Core Features:

- 메모 작성: 사용자가 텍스트, 이미지 또는 음성을 사용하여 메모를 작성할 수 있습니다.
- 메모 저장: 작성된 메모를 기기에 저장합니다.
- 메모 검색: 키워드를 기반으로 저장된 메모를 검색합니다.
- 카테고리 정리: 카테고리를 사용해 메모를 정리합니다.
- 배경 설정: 사용자가 원하는 배경 이미지를 업로드하여 화면의 특정 부분을 꾸밀 수 있도록 합니다.

## Style Guidelines:

- 배경색: 차분하고 편안한 느낌을 주는 연한 베이지색 (#F5F5DC).
- 주요 색상: 메모의 중요성을 강조하기 위해 따뜻한 주황색 (#FFB347).
- 액센트 색상: 상호 작용 요소에 대한 연한 파란색 (#ADD8E6).
- 본문 글꼴: 'PT Sans', 본문 내용에 적합한 산세리프체입니다.
- 제목 글꼴: 'Poppins', 짧은 머리글을 위한 기하학적 산세리프체입니다.
- 간결하고 직관적인 인터페이스를 위한 간단하고 최소한의 아이콘을 사용합니다.
- 내용의 명확성과 가독성을 극대화하기 위해 깨끗하고 정리된 레이아웃.
- 생산성을 높이고 사용자 상호 작용에 반응성을 부여하는 부드러운 애니메이션.
